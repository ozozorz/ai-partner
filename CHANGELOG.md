# Changelog

## 0.4.0 - 2026-07-21

- 实现 Rule-BT、LLM-Schema、Maid-IBC 与 A2“移除运行时监控与恢复”四个冻结变体；
- 增加 18 场景等效 GameTest 批处理，支持重复次数、批次 ID、原子检查点、断点恢复、独立终态/安全判定、IBCR 与扰动恢复统计；
- 增加模型、Prompt、温度、超时、安全边界、数据集、场景及实现哈希冻结锁，主实验启动前强制复核；
- 增加固定模型 72 指令离线评测器，支持限速、重试、断点恢复、保守费用上限，并输出 JVR、意图准确率、槽位 F1、CCR、URR 与 FRR；
- 补充预实验与 162/270 episodes 主实验命令、逐批次完整日志和工程质量冻结门槛。

## 0.3.0 - 2026-07-21

- 接通固定两阶段 `COLLECT_AND_DEPOSIT`，复用采集/存放状态机并共享父契约总超时；
- 持久化组合任务阶段、采集基线和存放进度，服务器重启后从正确阶段恢复；
- 加入 18 个冻结游戏内实验场景、21×8×21 有界测试区重置和三类显式运行中扰动；
- 为契约、验证、模型和场景日志补充批次、`episode_id`、`scenario_id`、期望终态、世界种子和维度；
- 随模组打包 72 条平衡中文离线指令集，并提供 Rule-BT 预测、分组指标和数据导出命令；
- 补充组合指令解析、场景注册表和冻结数据集一致性测试。

## 0.2.0 - 2026-07-21

- 修正 Alex 皮肤资源路径，并补全瘦臂护甲层和双手物品渲染；
- 将女仆储物区扩展为 36 格，接入四格原生护甲槽和副手槽；
- 添加潜行右键背包 UI、状态面板、跟随/待命/取消按钮和完整物品移动；
- 支持手持护甲普通右键直接穿戴，并安全归还被替换的旧护甲；
- 默认接入 `DEEPSEEK_API_KEY`、DeepSeek V4 Flash 和 JSON 输出；
- 参考 TouhouLittleMaid 的核心/任务行为分层，加入空闲漫步、开关门和更自然的延迟传送跟随。

## 0.1.0 - 2026-07-20

- 搭建 Minecraft 26.1.2、Java 25、Fabric Loader 0.19.3 工程；
- 添加使用官方 Alex 模型与贴图的可绑定女仆实体；
- 实现 `FOLLOW`、`STAY`、`COLLECT_BLOCK`、`DEPOSIT_ITEM` 和 `CANCEL`；
- 添加 TaskDefinition 注册表、IBC 编译器、目标谓词、有限重试和类型化失败码；
- 添加异步 OpenAI-compatible LLM 网关、严格 JobSpec Schema 和本地取消通道；
- 添加异步 JSONL 实验日志和服务器停止时落盘；
- 添加规则解析、JobSpec、任务注册表和契约生命周期单元测试。
